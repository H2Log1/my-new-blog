---

新建 `face_ws` 工作空间与 `src` 文件夹，在 `src` 文件夹下，新建功能包：
```bash
ros2 pkg create face_intefaces --dependencies sensor_msgs rosidl_default_generator --license Apache-2.0
```
删去 `src` 文件夹，新建 `srv` 文件夹，
在 `srv` 文件夹下，新建 `FaceDetector.srv` 文件，
在 `FaceDetector.srv` 文件中编写接口参数：
```
sensor_msgs/Image image
---
int16 number
float32 use_time
int32[] top
int32[] right
int32[] bottom
int32[] left
```
其中， `---` 上方为内容，下方为响应。
在 `CMakeLists.txt` 中加入：
```
rosidl_generate_interfaces(${PROJECT_NAME}
	"srv/FaceDetector.srv" # 路径
	DEPENDENCIES sensor_msgs # 依赖
)
```
在 `packages.xml` 中添加：
```
<member_of_group>rosidl_interface_package<\member_of_group>
```
终端转到 `src` 文件夹下，运行：
```bash
colcon build
source install/setup.bash # 如果终端使用的是bash，如果是zsh则换成对应后缀名文件
```
