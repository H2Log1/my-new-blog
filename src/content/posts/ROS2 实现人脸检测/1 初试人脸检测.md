下载必要模块
```bash
pip3 install face_recognition -i https://pypi.tuna.tsinghua.edu.cn/simple
```
在之前的工作空间 `service.ws` 下的 `src` 文件夹中，新建功能包：
```bash
ros2 pkg create demo_python_service --build-type ament_python --dependencies rclpy face_interfaces --license Apache-2.0
```
下载检测用图片 `defaut.jpg` 并存放在 `src/demo_python_service/resource` 路径下，在终端中输入：
```bash
colcon build
```
在 `setup.py` 中的 `data_files` 中添加：
```python
data_files=[
        ('share/ament_index/resource_index/packages',
            ['resource/' + package_name]),
        ('share/' + package_name, ['package.xml']),
        ('share/' + package_name+"/resource", ['resource/default.jpg']), # 新增部分
    ],
```
这样，图片会被拷贝一份到 `install/demo_python_service/share/demo_python_service/resource ` 下，便于后续读取文件，
`demo_python_service/demo_python_service` 下新建 `learn_face_detect.py` 文件：
```python
import face_recognition
import cv2
from ament_index_python.packages import get_package_share_directory # 获取功能包share目录绝对路径
import os

def main():
    defaut_image_path = os.path.join(
    	get_package_share_directory("demo_python_service"), 
    	"resource/default.jpg"
    	)
    print(f"图片真实路径：{defaut_image_path}")
    image = cv2.imread(defaut_image_path)
    face_locations = face_recognition.face_locations(image, 
    	number_of_times_to_upsample=1, 
    	model="hog")
    for top,right,bottom,left in face_locations:
        cv2.rectangle(image, (left, top), (right, bottom), (0, 0, 255), 4) # BGR
    cv2.imshow("Face Detecte Result", image)
    cv2.waitKey(0)
```
在 `setup.py` 中的 `entry_points` 下添加：
```python
entry_points={
        'console_scripts': [
            "learn_face_detect=demo_python_service.learn_face_detect:main", # 新增
        ],
```
在终端中运行并检验结果：
```bash
colcon build
source source install/setup.bash
ros2 run demo_python_service learn_face_detect
```
